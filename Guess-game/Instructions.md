# Number Guessing Game 🎯

A simple game where the computer selects a random number, and the player tries to guess it.

---

## Concepts Practiced
- Conditional statements (`if / else`)  
- Loops (`while`)  
- Random number generation  
- User input handling  

---

## How It Works
1. The computer generates a random number between 1 and 100.  
2. The user tries to guess the number.  
3. The program responds with:
   - `"Too high"` if the guess is above the number  
   - `"Too low"` if the guess is below the number  
4. Repeat steps 2–3 until the user guesses correctly.  

---

## What You'll Learn
- Program flow and logic  
- Decision-making with conditionals  
- Building a basic interactive game
