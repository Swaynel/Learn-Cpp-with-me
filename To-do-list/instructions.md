# To-Do List

A simple console-based task manager where the user can:

- Add new tasks  
- View existing tasks  
- Exit the program  

---

## How It Works
1. The program displays a menu with three options:
   1. Add Task  
   2. View Tasks  
   3. Exit  

2. The user selects an option by entering a number.  
3. Depending on the choice:
   - **Add Task:** Prompt the user to enter a new task.  
   - **View Tasks:** Display all tasks added so far.  
   - **Exit:** End the program.  

---

## Example Flow
```
Add Task
View Tasks
Exit
Choose option: 1
Enter task: Study C++
1. Add Task
2. View Tasks
3. Exit
Choose option: 2
Your Tasks:
1. Study C++
2. Add Task
3. View Tasks
4. Exit
Choose option: 3
```
