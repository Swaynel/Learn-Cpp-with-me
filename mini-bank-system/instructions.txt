This project simulates a very simple banking system where a user can:
Check balance
Deposit money
Withdraw money
Exit
It teaches:
Functions – to organize code
Variables – to track balance
User input and validation
Looping with menu options
Example Flow

```
Welcome to Mini Bank!

1. Check Balance
2. Deposit
3. Withdraw
4. Exit

Choose option: 2
Enter amount to deposit: 500
Deposit successful! Current balance: 500

Choose option: 3
Enter amount to withdraw: 200
Withdrawal successful! Current balance: 300
