# Mini Bank System

This project simulates a very simple banking system where a user can:

- Check balance  
- Deposit money  
- Withdraw money  
- Exit  

---

## Learning Objectives
This project helps you practice:

- **Functions** – Organize code into reusable blocks  
- **Variables** – Track and update the account balance  
- **User Input & Validation** – Handle user choices and amounts safely  
- **Looping & Menu Options** – Navigate through options until the user exits  

---
