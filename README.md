## What You Will Learn
- **Basics:** Variables, data types, operators, input/output (`cin`, `cout`)  
- **Control Flow:** `if`, `else`, `switch`, loops (`for`, `while`)  
- **Functions:** Declaration, parameters, return values, recursion  
- **Arrays & Strings:** 1D/2D arrays, `std::string`, string operations *(Coming Soon!)*

## Projects (Project-Based Learning with Code Samples)

### ✅ Completed
1. **Number Guessing Game** – Simple console game to guess numbers.  
2. **To-Do List** – Manage tasks using arrays and basic I/O.  

### 🚧 In Progress
3. **Mini Bank System** – Simulate basic banking operations.  
4. **Tic-Tac-Toe Game** – Two-player game using 2D arrays and loops with win-checking logic.  
5. **Simple Calculator** – Perform basic and scientific operations using functions and `<cmath>`.  
6. **Rock Paper Scissors** – Practice random generation, enums, control flow, and score tracking.  
7. **CGPA Calculator** – Input grades and credits to compute GPA using arrays and arithmetic.
